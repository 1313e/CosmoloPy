CosmoloPy

a cosmology package for Python.

Help out

Please join the developer discussion group and help turn this into a comprehensive community-maintained package of basic cosmology routines.

Download

You can clone the project with Git by running: 

$ git clone git://github.com/roban/CosmoloPy
or view the source code on GitHub: roban/CosmoloPy.

You can also download this project in either zip or tar formats.

Features

Current capabilities include
  • easy calculation of cosmological distance measures as a function of redshift and cosmological parameters,
  • perturbation-theory-based calculation of the matter power spectrum and halo collapse fraction,
  • integration of simple reionization models and calculation of the electron scattering (Thompson scattering) optical depth due to reionization.

Status

The project is currently very preliminary. Please join the developer discussion group if you are interested in using the code and ask the status of the routines you are interested in using. Please request new features if there's something you'd like to see added, or contribute your own code if you've written something new.
