CosmoloPy

a basic NumPy/SciPy-based cosmology package for Python.

Features

Current capabilities include easy calculation of cosmological distance measures as a function of redshift and cosmological parameters, perturbation-theory-based calculation of the matter power spectrum and halo collapse fraction, integration of simple reionization models and calculation of the electron scattering (Thompson scattering) optical depth due to reionization.

Status

The project is currently very preliminary. Please email me if you are interested in using the code and I can advise you on the current status.

Authors
