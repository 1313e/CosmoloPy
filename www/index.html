CosmoloPy

a cosmology package for Python.

Contribute

You can contribute by requesting new features, testing out the existing capabilities, or contributing your own code.

Please join the developer discussion group and help turn CosmoloPy into a comprehensive community-maintained package of basic cosmology routines. Whether you plan to be a casual user or frequent contributor, this is the place to introduce yourself and discuss the future of the package.

Status

The project is currently somewhat preliminary. We need more testing and more features.

Please join the developer discussion group if you are interested in using the code and ask about the status of the routines you are interested in using.

Download

You can clone the project with Git by running: 

$ git clone git://github.com/roban/CosmoloPy
or view the source code on GitHub: roban/CosmoloPy.

You can also download this project in either zip or tar formats.

Installation

  • Download the package (see above).
  • Unpack it.
  • Run the installation program:
python setup.py build
python setup.py nosetests
sudo python setup.py install

Features

Current capabilities include
  • calculation of cosmological distance measures as a function of redshift and cosmological parameters,
  • perturbation-theory-based calculation of the matter power spectrum and halo collapse fraction,
  • integration of simple reionization models and calculation of the electron scattering (Thompson scattering) optical depth due to reionization.

Acknowledgment

If you use any of the routines that make use of power spectrum calculations, please cite

Eisenstein, D. J., & Hu, W. 1999, ApJ, 511, 5 [astro-ph/9710252]
as the source of the matter power spectrum.

The use of CosmoloPy in general does not require any acknowledgment, but it is nice as a courtesy and as a service to others who might benefit from finding out about it. Please consider including the URL of this page: http://roban.github.com/CosmoloPy/
